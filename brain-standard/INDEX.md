## Storage Notes

Standards, specifications, templates, implementation documents, and operational checklists are distinct document types. Repository placement and index listing do not convert one type into another.

Implementation profile and setup documents support the Obsidian/Codex reference implementation only. They do not reduce software neutrality or make that implementation mandatory for compliant use.

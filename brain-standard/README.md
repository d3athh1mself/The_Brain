# The Brain Standard

The Brain is an implementation-independent Knowledge Operating System designed to preserve, organize, validate, and evolve knowledge across tools, agents, workflows, and storage systems.

## Current Deployment State

Status: Controlled bare usable deployment.

The repository may be used for supervised setup, controlled ingestion testing, and implementation preparation. It is not approved for unrestricted autonomous agent operation.

## Safe Agent Mode

Agents may inspect, summarize, classify, extract, draft, validate, propose metadata, propose relationships, record provenance, and prepare review outputs.

Agents must not approve, delete, publish, export, migrate, downgrade privacy classification, override validation failures, change Object Identity, or mark records authoritative without human review.
